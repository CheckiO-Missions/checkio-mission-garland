Dieser Wert bestimmt, wie viele benachbarte Positionen links und rechts von der Beleuchtung dieses Lichts erreicht werden, zusätzlich zur Beleuchtung der Position dieses Lichts selbst. (Ein Licht, dessen Helligkeit gleich Null ist, beleuchtet also nur sich selbst, hat aber keine Möglichkeit, seine Nachbarn zu beleuchten).

Ihre Aufgabe ist es, so wenig einzelne Lichter wie möglich einzuschalten, so dass jede Position der gesamten Reihe von mindestens einem Licht beleuchtet wird. Da die kleinste funktionierende Teilmenge von Lichtern nicht unbedingt eindeutig ist, sollte Ihre Funktion nur die Anzahl der Lichter zurückgeben, die zur Beleuchtung der gesamten Reihe erforderlich sind.
